# 📚 Estudos de HTML5 e CSS3

Repositório criado para organizar meus exercícios e aprendizados do curso de **HTML5 e CSS3** do canal Curso em Vídeo (Prof. Gustavo Guanabara).

## 🗂 Estrutura do Repositório

O projeto está dividido em pastas para facilitar a navegação:

- **/Html**: Contém todos os arquivos de marcação e exercícios de estrutura.
- **/Css**: Contém as folhas de estilo para estilização dos exercícios.

---

## 🛠 Tecnologias Utilizadas

* **HTML5**: Marcação de conteúdo.
* **CSS3**: Estilização e layout.
* **VS Code**: Editor de código.
* **GitHub Desktop**: Gerenciamento de versão.
